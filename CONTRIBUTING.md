# Contributing to aifed

Thank you for your interest in contributing to aifed! This document outlines the various ways you can contribute to the project.

## Ways to Contribute

### Report Issues

Don't be shy! Whether you found a bug, have a question, or want to request a feature, we welcome all feedback.

- Open an issue on GitHub
- Describe the problem or idea as clearly as you can

### Spread the Word

Help grow the community by discussing this project with others, share the project with friends who might find it useful.
A larger community means more feedback, more contributions, and a better tool for everyone.

### Code Development

Interested in contributing code? Check out the [Development Guide](docs/development-guide.md) for:

- Environment setup instructions
- Development workflow
- Agent action wizards for common tasks
